# How Many Bro PRD

## 목표

투자 손실과 수익을 차가운 원화 숫자 대신 생활감 있는 물품 수량으로 바꿔 사용자가 자신의 상태를 빠르게 이해하고 공유할 수 있게 한다.

## 핵심 사용자 흐름

### 1차 MVP

1. 사용자가 본인의 손익 금액을 입력한다. 손실은 음수로 입력한다.
2. 사용자가 입력 버튼을 누른다.
3. 앱이 결과 화면으로 전환한다.
4. 앱이 손익 금액을 금액대별 물품 수량으로 환산해 보여준다.

### 2차 MVP

1. 국내/미국 종목 검색을 제공한다.
2. 최근 종가 기준으로 평가손익을 계산한다.
3. 공공데이터포털, 미국 일봉 데이터, 또는 토스증권 API를 서버/배치에서 수집한다.
4. 익명 표본 기반으로 비슷한 사용자 수, 평균 손실, 분위 정보를 보여준다.
5. 광고가 붙는 경우 데이터 출처, 비공식 서비스, 투자 조언 아님 고지를 명확히 표시한다.

## 디자인 원칙

- 첫 화면은 랜딩 페이지가 아니라 계산기다.
- 첫 MVP에서는 입력 버튼 하나만 둔다.
- 가장 큰 비유 하나를 먼저 보여준다.
- 환산 기준은 짧게라도 항상 노출한다.
- 재미있는 문구가 숫자의 정확성을 가리지 않게 한다.

## 다음 확장 후보

- 종목 검색과 최근 종가 조회 API
- 서버 기반 익명 집계
- 결과 공유 이미지 생성
- 종목별 커뮤니티 분포와 기간 필터
- 손실/수익 구간별 다른 캐릭터 연출
